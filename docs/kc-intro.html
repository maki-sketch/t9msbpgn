科创部 (STEM Department)

天津市第九中学学生社团核心部门

部门宗旨：
激发科学兴趣，培养创新思维，动手改变世界。在这里，代码是我们的语言，创造是我们的信仰。

我们要在这个部门做什么？

科创部致力于为同学们提供最前沿的科技体验平台，主要涉及领域包括：

🤖 机器人技术

学习 Arduino 与树莓派开发，设计并组装智能避障小车与机械臂。

💻 编程算法

Python 数据分析、C++ 算法竞赛（NOIP）辅导以及网页开发基础。

🖨️ 3D 打印与设计

使用 CAD 软件进行三维建模，并将你的创意通过 3D 打印机变为现实。

🚁 无人机航拍

无人机操控技巧训练，校园全景航拍与视频剪辑。

过往荣誉

  • 全国青少年科技创新大赛 - 多个市级一等奖
  • 天津市中小学机器人竞赛 - 最佳创意奖
  • 校内科技节 - 优秀组织部门

加入我们

无论你是编程大神，还是对科技充满好奇的萌新，科创部都欢迎你的加入！
